Sobre o Projeto

Gas Mais Barato - Dashboard de Preços GLP

Uma aplicação web desenvolvida para facilitar a consulta de preços oficiais de GLP (Gás Liquefeito de Petróleo) de 13kg em diferentes cidades brasileiras, utilizando dados da ANP (Agência Nacional do Petróleo, Gás Natural e Biocombustíveis).

Tecnologias Utilizadas

  • Backend: Flask (Python)
  • Frontend: HTML, CSS, JavaScript
  • Framework CSS: Bootstrap 5
  • Processamento de Dados: Pandas
  • Dados: CSV oficial da ANP

Funcionalidades

  • Busca por cidade e estado
  • Visualização de preços oficiais
  • Informações das revendedoras
  • Filtros avançados
  • Interface responsiva
  • API REST para integração

Fonte dos Dados

Todos os dados utilizados nesta aplicação são oficiais e fornecidos pela ANP (Agência Nacional do Petróleo, Gás Natural e Biocombustíveis), órgão responsável pela regulação e fiscalização do setor de combustíveis no Brasil.

Atualização dos Dados

Os dados são atualizados regularmente conforme disponibilização da ANP. O período atual abrange as últimas 4 semanas de coleta de preços.

Transparência

Este projeto é desenvolvido com total transparência e utiliza apenas dados públicos e oficiais. Não há coleta de dados pessoais dos usuários.

Objetivo

O objetivo principal é ajudar os brasileiros a encontrar o melhor preço do GLP em sua região, promovendo transparência e facilitando o acesso às informações oficiais de preços.

Dica: Use os filtros de busca para encontrar preços específicos da sua cidade ou região. Os dados são atualizados regularmente com informações oficiais da ANP.
